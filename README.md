#  SAVIN

Savin adalah sebuah aplikasi yang digunakan sebagai semacam e-wallet yang berguna untuk menabung dengan batas waktu tertentu.

Project ini menggunakan Clean Architecture dan sudah menggunakan MVVM design pattern-> Model, View Model , View Model.
Untuk library dan alat yang digunakan yaitu Dagger hilt untuk dependency injection, Retrofit untuk Pemanggilan API Enpoint, Glide untuk Pemanggilan Gambar dan Lottie untuk animasi Loading.

##  Fitur    
1. Login --> PIN : 123456
2. Melihat Dompet yang sudah tersedia
3. Melihat transaksi pada setiap dompet
4. Melihat seluruh transaksi pada akun tersebut
5. Search Transaksi

##  Implementasi konsep dan Teknologi
- **Clean Architecture** Agar pembacaan kode dapat lebih mudah
- **MVVM Design Pattern** Agar mudah dalam mencari error dan ketika melakukan test
- **Dagger Hilt** Agar mudah ketika ingin melakukan injection dan fleksibel
- **Retrofit** Agar pada saat pemanggilan API mudah 
- **Glide** Agar pada saat pemanggilan image dari luar, lebih mudah dan halus
- **Lottie** Agar pada saat loading memiliki animasi yang dapat dilihat
